# Array List vs Linked List
| Metric                                | Array List    | Linked List   |
| -----------                           | -----------   | -----------   |
| Finding an Element                    |     O(1)      |   O(N)        |               
| Insertion/Deletion at Beginning       |     O(N)      |   O(1)        |
| Insertion/Deletion at Arbitrary Point |     O(N)      |   O(N)        |
